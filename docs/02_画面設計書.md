# 画面設計書

## 1. 画面一覧

| 画面名 | URL |
|--------|-----|
| 映画一覧 | `/` |
| 映画詳細 | `/movies/:id` |
| TMDB検索 | `/movies/search` |
| TMDB検索結果 | `/movies/search/results` |
| 映画登録 | `/movies/new` |
| 映画編集 | `/movies/:id/edit` |

---

## 2. 画面設計

### 映画一覧画面　`/`

**表示項目**

| 項目 | 取得元 |
|------|--------|
| ポスター画像 | movies.poster_url |
| タイトル | movies.title |
| 評価 | viewing_history.rating |
| 登録日 | viewing_history.created_at |
| タイトル検索窓 | ユーザー入力 |
| ユーザーアイコン | プレースホルダー（将来: users.icon） |
| ユーザー名       | プレースホルダー（将来: users.name） |
| 空状態メッセージ | 固定文言（登録件数が0件のとき表示）「登録された映画がありません」 |

**操作**

| 操作 | 説明 |
|------|------|
| タイトル検索 | 検索窓にキーワードを入力して絞り込み |
| カードをクリック | 映画詳細画面へ遷移 |
| 映画を登録するボタン | TMDB検索画面へ遷移 |

**画面遷移**

```
映画一覧 ──カードクリック──→ 映画詳細
         ──登録ボタン──────→ TMDB検索
```

---

### 映画詳細画面　`/movies/:id`

**表示項目**

| 項目 | 取得元 |
|------|--------|
| ポスター画像 | movies.poster_url |
| タイトル | movies.title |
| 公開年 | movies.release_year |
| 概要 | movies.overview |
| 評価 | viewing_history.rating |
| レビュー | viewing_history.review |
| 登録日 | viewing_history.created_at |

**操作**

| 操作 | 説明 |
|------|------|
| 編集ボタン | 映画編集画面へ遷移 |
| 削除ボタン | 削除確認ダイアログを表示 |
| ダイアログ - 確認ボタン | 映画を削除して映画一覧へ遷移 |
| ダイアログ - キャンセル | ダイアログを閉じる |
| 戻るボタン | 映画一覧画面へ遷移 |

**画面遷移**

```

映画詳細 ──編集ボタン──→ 映画編集
　　　　　削除ボタン ──→ 削除確認ダイアログ ──確認──→ 映画一覧
　　　　　　　　　　　　　　　　　　　　　 ──キャンセル──→ （ダイアログを閉じる）
　　　　　　──戻るボタン──→ 映画一覧　　　　　
```

---

### TMDB検索画面　`/movies/search`

**表示項目**

| 項目 | 取得元 |
|------|--------|
| 検索窓 | ユーザー入力 |

**操作**

| 操作 | 説明 |
|------|------|
| キーワード入力→検索ボタン | TMDB検索結果画面へ遷移 |
| 戻るボタン | 映画一覧画面へ遷移 |

**画面遷移**

```
TMDB検索 ──検索ボタン──→ TMDB検索結果
         ──戻るボタン──→ 映画一覧
```

---

### TMDB検索結果画面　`/movies/search/results`

**表示項目**

| 項目 | 取得元 |
|------|--------|
| ポスター画像 | TMDB API |
| タイトル | TMDB API |
| 公開年 | TMDB API |
| 概要 | TMDB API |
| 空状態メッセージ | 固定文言（検索結果が0件のとき表示）「該当する映画が見つかりませんでした」 |

**操作**

| 操作 | 説明 |
|------|------|
| 映画を選択するボタン | 映画登録画面へ遷移（選択した映画の情報を引き継ぐ） |
| 検索へ戻るボタン | TMDB検索画面へ遷移 |
| 映画一覧へ戻るボタン | 映画一覧画面へ遷移 |
**画面遷移**

```
TMDB検索結果 ──映画を選択──→ 映画登録
             ──検索へ戻るボタン──→ TMDB検索
             　──映画一覧へ戻るボタン──→ 映画一覧
```

---

### 映画登録画面　`/movies/new`

**表示項目**

| 項目 | 取得元 |
|------|--------|
| ポスター画像（確認用） | TMDB API（前画面から引き継ぎ） |
| タイトル（確認用） | TMDB API（前画面から引き継ぎ） |
| 評価入力フォーム | ユーザー入力（必須・0.1〜5.0） |
| レビュー入力フォーム | ユーザー入力（任意） |

**操作**

| 操作 | 説明 |
|------|------|
| 登録ボタン | DBに保存して映画一覧画面へ遷移 |
| キャンセルボタン | キャンセル確認ダイアログを表示 |
| ダイアログ - 確認ボタン | TMDB検索結果画面へ戻る |
| ダイアログ - キャンセル | ダイアログを閉じる |

**画面遷移**

```
映画登録 ──登録ボタン────────────────────→ 映画一覧
         ──キャンセルボタン──→ 確認ダイアログ ──確認──→ TMDB検索結果
                                            ──キャンセル──→ （ダイアログを閉じる）
```

---

### 映画編集画面　`/movies/:id/edit`

**表示項目**

| 項目 | 取得元 |
|------|--------|
| タイトル（変更不可） | movies.title |
| 評価入力フォーム | viewing_history.rating（現在値を初期表示） |
| レビュー入力フォーム | viewing_history.review（現在値を初期表示） |

**操作**

| 操作 | 説明 |
|------|------|
| 更新ボタン | DBを更新して映画詳細画面へ遷移 |
| キャンセルボタン | キャンセル確認ダイアログを表示 |
| ダイアログ - 確認ボタン | 映画詳細画面へ戻る |
| ダイアログ - キャンセル | ダイアログを閉じる |

**画面遷移**

```
映画編集 ──更新ボタン────────────────────→ 映画詳細
         ──キャンセルボタン──→ 確認ダイアログ ──確認──→ 映画詳細
                                            ──キャンセル──→ （ダイアログを閉じる）
```

## 3. 文言定義

### ダイアログ文言

| ダイアログ | タイトル | 本文 | 確認ボタン | キャンセルボタン |
|-----------|---------|------|-----------|----------------|
| 削除確認 | 「映画を削除」 | 「この映画を削除しますか？この操作は取り消せません。」 | 「削除する」 | 「キャンセル」 |
| 登録キャンセル確認 | 「登録を中止」 | 「入力内容が破棄されます。よろしいですか？」 | 「中止する」 | 「戻る」 |
| 編集キャンセル確認 | 「編集を中止」 | 「変更内容が破棄されます。よろしいですか？」 | 「中止する」 | 「戻る」 |

---

### 空状態メッセージ

| 画面 | 発生条件 | メッセージ |
|------|---------|-----------|
| 映画一覧 | 登録件数が0件 | 「登録された映画がありません」 |
| TMDB検索結果 | 検索結果が0件 | 「該当する映画が見つかりませんでした」 |

---

### バリデーションエラーメッセージ

| 項目 | 条件 | メッセージ |
|------|------|-----------|
| 評価（rating） | 未入力 | 「評価を入力してください」 |
| 評価（rating） | 0.1〜5.0の範囲外 | 「評価は0.1〜5.0で入力してください」 |
| TMDB検索キーワード | 未入力 | 「検索キーワードを入力してください」 |
