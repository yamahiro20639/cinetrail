# API詳細設計書

## 1. 共通仕様

### ベースURL
```
http://localhost:8080/api
```

### 共通レスポンス形式（エラー時）

```json
{
  "status": 404,
  "message": "映画が見つかりません"
}
```

### 使用するHTTPステータスコード

| コード | 意味 | 使用場面 |
|--------|------|---------|
| 200 | OK | 取得・更新成功 |
| 201 | Created | 登録成功 |
| 204 | No Content | 削除成功 |
| 400 | Bad Request | バリデーションエラー |
| 404 | Not Found | 対象データが存在しない |
| 500 | Internal Server Error | サーバー内部エラー |

---

## 2. API定義

### GET /api/movies　映画一覧取得

**概要**：登録済み映画の一覧を取得する。タイトルで絞り込み検索も可能。

**リクエスト**

| 種別 | パラメータ名 | 型 | 必須 | 説明 |
|------|------------|-----|------|------|
| クエリパラメータ | title | String | 任意 | タイトルで部分一致検索 |

リクエスト例：
```
GET /api/movies
GET /api/movies?title=インセプション
```

**レスポンス（200 OK）**

```json
[
  {
    "id": 1,
    "title": "インセプション",
    "posterUrl": "https://image.tmdb.org/t/p/w500/xxx.jpg",
    "releaseYear": 2010,
    "rating": 4.5,
    "createdAt": "2026-05-18T10:00:00"
  },
  {
    "id": 2,
    "title": "タイタニック",
    "posterUrl": "https://image.tmdb.org/t/p/w500/yyy.jpg",
    "releaseYear": 1997,
    "rating": 3.5,
    "createdAt": "2026-05-17T09:00:00"
  }
]
```

> 一覧画面はカード表示のため、概要（overview）は含まない。

**バリデーション**
- なし（全件取得・部分一致検索どちらも制限なし）

### GET /api/movies/:id　映画詳細取得

**概要**：登録済み映画の詳細情報を取得する

**リクエスト**

| 種別 | パラメータ名 | 型 | 必須 | 説明 |
|------|------------|-----|------|------|
|  パスパラメーター| id |  INT|  必須|viewing_historyのID |

リクエスト例：
```
GET /api/movies/1
```

**レスポンス（200 OK）**

```json

{
  "id": 1,
  "title": "インセプション",
  "overview": "夢の中に侵入して情報を盗む...",
  "posterUrl": "https://image.tmdb.org/t/p/w500/xxx.jpg",
  "releaseYear": 2010,
  "rating": 4.5,
  "review": "映像が美しかった",
  "createdAt": "2026-05-18T10:00:00"
}

```

**エラーレスポンス**

| ステータス | 発生条件 | メッセージ例 |
|-----------|---------|------------|
| 400 | idが数値以外 | "IDは数値で指定してください" |
|  404|  リクエストするidがviewing_historyに存在しない|映画が見つかりません |

**バリデーション**
- `id`：必須・数値のみ

### POST /api/movies　映画登録

**概要**：TMDBから取得した映画情報と、ユーザーの評価・レビューをDBに保存する。

**リクエスト**

| 種別 | パラメータ名 | 型 | 必須 | 説明 |
|------|------------|-----|------|------|
| ボディ | tmdbId | Integer | 必須 | TMDBの映画ID |
| ボディ | title | String | 必須 | 映画タイトル |
| ボディ | overview | String | 任意 | 映画概要 |
| ボディ | posterUrl | String | 任意 | ポスター画像URL |
| ボディ | releaseYear | Integer | 任意 | 公開年 |
| ボディ | rating | Decimal | 必須 | 評価（0.1〜5.0） |
| ボディ | review | String | 任意 | レビュー |

リクエスト例：
```json
{
  "tmdbId": 1,
  "title": "インセプション",
  "overview": "夢の中に侵入して情報を盗む...",
  "posterUrl": "https://image.tmdb.org/t/p/w500/xxx.jpg",
  "releaseYear": 2010,
  "rating": 4.5,
  "review": "映像が美しかった"
}
```

**レスポンス（201 Created）**

```json
{
  "id": 1,
  "title": "インセプション",
  "rating": 4.5,
  "createdAt": "2026-05-18T10:00:00"
}
```

**エラーレスポンス**

| ステータス | 発生条件 | メッセージ例 |
|-----------|---------|------------|
| 400 | tmdbIdが未入力 | "tmdbIdは必須です" |
| 400 | ratingが未入力または範囲外 | "評価は0.1〜5.0で入力してください" |
| 400 | ratingが数値以外 | "評価は数字で入力してください" |
| 400 | titleが未入力 | "タイトルは必須です" |
| 400 | titleが101文字以上 | "タイトルは100文字以内で入力してください" |
| 400 | reviewが1001文字以上 | "レビューは1000文字以内で入力してください" |

**バリデーション**
- `tmdbId`：必須・数値のみ
- `title`：必須・100文字以内
- `rating`：必須・数値のみ・0.1以上5.0以下
- `review`：任意・1000文字以内

### PUT /api/movies/:id　映画更新

**概要**：登録済み映画の評価・レビューを更新する。

**リクエスト**

| 種別 | パラメータ名 | 型 | 必須 | 説明 |
|------|------------|-----|------|------|
| パスパラメータ | id | Integer | 必須 | viewing_historyのID |
| ボディ | rating | Decimal | 必須 | 評価（0.1〜5.0） |
| ボディ | review | String | 任意 | レビュー |

リクエスト例：
```
PUT /api/movies/1
```
```json
{
  "rating": 5.0,
  "review": "やっぱり最高だった"
}
```

**レスポンス（200 OK）**

```json
{
  "id": 1,
  "title": "インセプション",
  "rating": 5.0,
  "review": "やっぱり最高だった",
  "createdAt": "2026-05-18T10:00:00"
}
```

**エラーレスポンス**

| ステータス | 発生条件 | メッセージ例 |
|-----------|---------|------------|
| 400 | idが数値以外 | "IDは数値で指定してください" |
| 400 | ratingが未入力 | "評価を入力してください" |
| 400 | ratingが範囲外 | "評価は0.1〜5.0で入力してください" |
| 400 | ratingが数値以外 | "評価は数字で入力してください" |
| 400 | reviewが1001文字以上 | "レビューは1000文字以内で入力してください" |
| 404 | 対象IDが存在しない | "映画が見つかりません" |

**バリデーション**
- `id`：必須・数値のみ
- `rating`：必須・数値のみ・0.1以上5.0以下
- `review`：任意・1000文字以内

### GET /api/tmdb/search　TMDB検索

**概要**：タイトルまたは人物名でTMDBを検索し、候補一覧を返す。

**リクエスト**

| 種別 | パラメータ名 | 型 | 必須 | 説明 |
|------|------------|-----|------|------|
| クエリパラメータ | query | String | 必須 | 検索キーワード |

リクエスト例：
```
GET /api/tmdb/search?query=インセプション
```

**レスポンス（200 OK）**

```json
[
  {
    "tmdbId": 27205,
    "title": "インセプション",
    "overview": "夢の中に侵入して情報を盗む...",
    "posterUrl": "https://image.tmdb.org/t/p/w500/xxx.jpg",
    "releaseYear": 2010
  }
]
```

**エラーレスポンス**

| ステータス | 発生条件 | メッセージ例 |
|-----------|---------|------------|
| 400 | queryが未入力または空白のみ | "検索キーワードを入力してください" |

**バリデーション**
- `query`：必須・空白のみ不可
