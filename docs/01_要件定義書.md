# 要件定義書

## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| アプリ名 | CineTrail |
| 目的 | 映画の視聴履歴を管理・記録できるWebサービス |
| 参考サービス | Filmarks |
| 作成日 | 2026-05-11 |

---

## 2. 技術スタック

| 種別 | 技術 |
|------|------|
| フロントエンド | React |
| バックエンド | Spring Boot |
| データベース | MySQL |
| 外部API | TMDB（The Movie Database） |
| バージョン管理 | GitHub |
| インフラ | Render（フロント・バックエンド）、Railway（MySQL） |
| CI/CD | GitHub × Render 自動デプロイ（フェーズ⑥で実装） |

---

## 3. ユーザー種別

| 種別 | 説明 | 今回 | 将来 |
|------|------|------|------|
| 未認証ユーザー | ログインしていないユーザー | 映画一覧・詳細の閲覧のみ | 同左 |
| 認証ユーザー | ログイン済みユーザー | 全機能利用可（認証は将来対応） | レビュー・ウォッチリスト等 |

> 今回は認証なしで実装し、将来の認証機能追加に備えた設計を行う。

---

## 4. 機能要件

### 今回のスコープ

| 機能 | 内容 | 備考 |
|------|------|------|
| 映画登録 | TMDB検索→DB保存・評価（必須）・レビュー（任意）を記録 | TMDBから取得：タイトル・概要・ポスター・公開年・キャンセル確認ダイアログを表示 |
| 映画一覧 | 自分が登録した映画をカード形式で表示 | インスタ風レイアウト・ユーザー情報はプレースホルダー |
| 映画詳細 | タイトル・概要・評価・レビューを表示 | |
| 映画編集 | 登録済み映画の評価・レビューを更新 | キャンセル確認ダイアログを表示 |
| 映画削除 | 登録済み映画を削除 | 削除確認ダイアログを表示 |
| 映画検索（登録済み） | 自分が登録した映画をタイトルで部分一致検索 | 映画一覧画面の検索窓から操作 |
| 映画検索（TMDB） | TMDBに対してタイトル・人物名で検索 | 映画登録時に使用 |

### 将来対応

| 機能 | 内容 |
|------|------|
| ユーザー認証 | ログイン・ログアウト・ユーザー登録 |
| ユーザープロフィール | アイコン・ユーザー名の表示・管理 |
| ユーザー検索 | 他ユーザーのプロフィール・視聴履歴を閲覧 |
| タイムライン | 全ユーザーのレビューを表示する公開フィード |
| ウォッチリスト | 「観たい」映画の登録（公開済み・公開前問わず） |
| コミュニティ | 映画好き同士のオープンチャット（WebSocket） |
| 人物データ管理 | 出演者・監督のDB保存・人物名での登録済み映画検索 |

---

## 5. 非機能要件

| 項目 | 内容 |
|------|------|
| セキュリティ | 将来の認証に備えてパスワードはハッシュ化設計 |
| 検索方式 | 部分一致検索（LIKE句） |
| 評価方式 | 0.1〜5.0点（DECIMAL(3,1)） |
| 無料枠運用 | GitHub・Render・Railwayの無料枠内で運用 |

---

## 6. データ要件

### 映画登録時の送受信データ

| データ項目 | 取得元 |
|-----------|-------|
| タイトル | TMDB |
| 概要 | TMDB |
| ポスター画像URL | TMDB |
| 公開年 | TMDB |
| TMDB ID | TMDB |
| 評価（rating） | ユーザー入力（必須・0.1〜5.0） |
| レビュー | ユーザー入力（任意） |

### テーブル構成（概要）

| テーブル名 | 役割 |
|-----------|------|
| movies | 映画カタログ（共有・TMDB情報を保存） |
| viewing_history | 個人の視聴履歴（評価・レビューを含む） |
| users（将来） | ユーザー情報 |
| watchlist（将来） | ウォッチリスト |
| people（将来） | 出演者・監督情報 |
| movie_people（将来） | 映画と人物の中間テーブル（多対多） |
