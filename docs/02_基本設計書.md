# 基本設計書

## 1. システム構成図

```
[ブラウザ]
    ↕ HTTP
[React（フロントエンド）] ← Render
    ↕ REST API
[Spring Boot（バックエンド）] ← Render
    ↕ JDBC
[MySQL（DB）] ← Railway

[Spring Boot] ↔ [TMDB API（外部）]

※ GitHubへのpushでRenderが自動デプロイ（CI/CD）
```

---

## 2. 画面一覧

| ページ名 | URL | 内容 |
|---------|-----|------|
| 映画一覧 | `/` | 自分が登録した映画をカード形式で表示・検索窓あり |
| 映画詳細 | `/movies/:id` | タイトル・概要・評価・レビュー表示・削除ダイアログ |
| TMDB検索 | `/movies/search` | タイトル・人物名を入力してTMDB検索 |
| TMDB検索結果 | `/movies/search/results` | TMDB検索結果の候補一覧から映画を選択 |
| 映画登録 | `/movies/new` | 評価・レビューを入力してDBに保存 |
| 映画編集 | `/movies/:id/edit` | 登録済み映画の評価・レビューを更新 |

## 3. REST API一覧

### 映画API

| メソッド | エンドポイント | 内容 |
|---------|--------------|------|
| GET | `/api/movies` | 映画一覧取得 |
| GET | `/api/movies?title=` | 登録済み映画をタイトルで部分一致検索 |
| GET | `/api/movies/:id` | 映画詳細取得 |
| POST | `/api/movies` | 映画登録 |
| PUT | `/api/movies/:id` | 映画更新 |
| DELETE | `/api/movies/:id` | 映画削除 |

### TMDB API

| メソッド | エンドポイント | 内容 |
|---------|--------------|------|
| GET | `/api/tmdb/search?query=` | TMDBに対してタイトル・人物名で検索 |

### POST /api/movies リクエストボディ

```json
{
  "tmdbId": 12345,
  "title": "インセプション",
  "overview": "夢の中に侵入して...",
  "posterUrl": "https://image.tmdb.org/...",
  "releaseYear": 2010,
  "rating": 4.5,
  "review": "映像が美しかった"
}
```

## 4. 技術選定理由

| 技術 | 選定理由 |
|------|---------|
| Spring Boot | 現場での採用率が高く、学習価値が高い |
| React | コンポーネント設計の学習に最適 |
| MySQL | リレーショナルDB設計の基礎を学べる |
| TMDB API | 無料・豊富な映画データ・REST APIの学習に最適 |
| Render | 無料枠でSpring Boot・Reactのデプロイが可能 |
| Railway | 無料枠でMySQLのホスティングが可能 |
