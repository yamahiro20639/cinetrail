# クラス設計書

## 1. パッケージ構成

```
com.example.cinetrail
├── controller
│   ├── MovieController.java
│   └── TmdbController.java
├── service
│   ├── MovieService.java
│   └── TmdbService.java
├── repository
│   ├── MovieRepository.java
│   └── ViewingHistoryRepository.java
├── entity
│   ├── Movie.java
│   └── ViewingHistory.java
├── dto
│   ├── request
│   │   ├── MovieRequest.java
│   │   └── MovieUpdateRequest.java
│   └── response
│       ├── MovieListResponse.java
│       ├── MovieDetailResponse.java
│       ├── MovieCreateResponse.java
│       ├── MovieUpdateResponse.java
│       └── TmdbMovieResponse.java
└── exception
    ├── GlobalExceptionHandler.java
    ├── ErrorResponse.java
    └── MovieNotFoundException.java
```

---

## 2. クラス一覧

| 層 | クラス名 | 役割 |
|----|---------|------|
| Controller | MovieController | 映画関連APIのHTTPリクエストを受け取り、Serviceに処理を委譲する |
| Controller | TmdbController | TMDB検索APIのHTTPリクエストを受け取り、Serviceに処理を委譲する |
| Service | MovieService | 映画の登録・取得・更新・削除のビジネスロジックを担う |
| Service | TmdbService | TMDB APIを呼び出し、検索結果を返す |
| Repository | MovieRepository | moviesテーブルへのDB操作を担う |
| Repository | ViewingHistoryRepository | viewing_historyテーブルへのDB操作を担う |
| Entity | Movie | moviesテーブルに対応するクラス |
| Entity | ViewingHistory | viewing_historyテーブルに対応するクラス |
| DTO（Request） | MovieRequest | POST /api/movies のリクエストボディ |
| DTO（Request） | MovieUpdateRequest | PUT /api/movies/:id のリクエストボディ |
| DTO（Response） | MovieListResponse | GET /api/movies のレスポンス（1件分） |
| DTO（Response） | MovieDetailResponse | GET /api/movies/:id のレスポンス |
| DTO（Response） | MovieCreateResponse | POST /api/movies のレスポンス |
| DTO（Response） | MovieUpdateResponse | PUT /api/movies/:id のレスポンス |
| DTO（Response） | TmdbMovieResponse | GET /api/tmdb/search のレスポンス（1件分） |
| Exception | GlobalExceptionHandler | 全APIの例外を一括でハンドリングする |
| Exception | ErrorResponse | エラーレスポンスのデータ構造 |
| Exception | MovieNotFoundException | 映画が見つからない場合の例外（404） |
| Exception | DuplicateMovieException | 同じtmdbIdの映画がすでに登録済みの場合の例外（409） |

## 3. クラス詳細

### Controller層

#### MovieController

| 項目 | 内容 |
|------|------|
| アノテーション | `@RestController`, `@RequestMapping("/api/movies")` |
| 依存 | MovieService |

| メソッド名 | HTTPメソッド | パス | 説明 |
|-----------|------------|------|------|
| getMovies | GET | `/api/movies` | 映画一覧取得（titleクエリで絞り込み） |
| getMovie | GET | `/api/movies/{id}` | 映画詳細取得 |
| createMovie | POST | `/api/movies` | 映画登録 |
| updateMovie | PUT | `/api/movies/{id}` | 映画更新 |
| deleteMovie | DELETE | `/api/movies/{id}` | 映画削除 |

#### TmdbController

| 項目 | 内容 |
|------|------|
| アノテーション | `@RestController`, `@RequestMapping("/api/tmdb")` |
| 依存 | TmdbService |

| メソッド名 | HTTPメソッド | パス | 説明 |
|-----------|------------|------|------|
| searchMovies | GET | `/api/tmdb/search` | TMDBキーワード検索 |

---

### Service層

#### MovieService

| 項目 | 内容 |
|------|------|
| アノテーション | `@Service` |
| 依存 | MovieRepository, ViewingHistoryRepository |

| メソッド名 | 戻り値 | 説明 |
|-----------|--------|------|
| getMovies | `List<MovieListResponse>` | 全件取得またはタイトル部分一致検索 |
| getMovie | `MovieDetailResponse` | ID指定で1件取得。存在しない場合はMovieNotFoundExceptionをスロー |
| createMovie | `MovieCreateResponse` | 映画とviewing_historyを保存 |
| updateMovie | `MovieUpdateResponse` | viewing_historyの評価・レビューを更新 |
| deleteMovie | `void` | viewing_historyとmoviesを削除 |

#### TmdbService

| 項目 | 内容 |
|------|------|
| アノテーション | `@Service` |
| 依存 | RestTemplate（TMDB API呼び出し用） |

| メソッド名 | 戻り値 | 説明 |
|-----------|--------|------|
| searchMovies | `List<TmdbMovieResponse>` | TMDB APIにキーワードを送り、結果を返す |

---

#### ViewingHistoryRepository

| 項目 | 内容 |
|------|------|
| 継承 | `JpaRepository<ViewingHistory, Integer>` |

### DTO層

| フィールド名 | 型 | バリデーション |
|------------|-----|--------------|
| tmdbId | Integer | 必須・数値のみ |
| title | String | 必須・100文字以内 |

| releaseYear | Integer | 任意 |

| フィールド名 | 型 | バリデーション |
|------------|-----|--------------|

### 例外処理

#### GlobalExceptionHandler

| 項目 | 内容 |
|------|------|
| アノテーション | `@RestControllerAdvice` |

| ハンドル対象 | HTTPステータス | 説明 |
|------------|--------------|------|
| MovieNotFoundException | 404 | 映画が見つからない場合 |
| DuplicateMovieException | 409 | 同じtmdbIdの映画がすでに登録済みの場合 |
| MethodArgumentNotValidException | 400 | リクエストボディのバリデーションエラー（@Validによる） |
| MethodArgumentTypeMismatchException | 400 | パスパラメータの型変換失敗（idに数値以外が来た場合） |
| HttpMessageNotReadableException | 400 | リクエストボディの型変換失敗（ratingに文字列が来た場合など） |
| Exception | 500 | 予期しないサーバーエラー |

#### ErrorResponse

| フィールド名 | 型 | 説明 |
|------------|-----|------|
| status | Integer | HTTPステータスコード |
| message | String | エラーメッセージ |

#### MovieNotFoundException

| 項目 | 内容 |
|------|------|
| 継承 | `RuntimeException` |
| 用途 | 指定IDの映画が存在しない場合にServiceからスローし、GlobalExceptionHandlerで404として返す |
